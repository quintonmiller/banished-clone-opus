Changelog

A summary of what changed in each version of Exiled.

v0.6.0 — Current

  • Added comprehensive achievement system with 65 achievements across 7 categories, tracked persistently across games
  • Achievements viewable from Start Screen and Pause Menu with category filtering and scroll
  • 10 achievements grant per-game bonuses (migrated from the old milestone system)
  • 10 secret achievements with hidden descriptions until unlocked
  • Cross-game statistics tracking (games played, years survived, citizens born/died, buildings built, trades completed)

v0.5.0

  • Revamped festivals into multi-phase interactive fairs with group activities, traveling visitors, prosperity scoring, and reward choices
  • Added coordinated group activities at fairs: races, group dances, feast circles, tug of war, parades, scavenger hunts
  • Overhauled relationship system with organic score-based progression, partnership, marriage, divorce, and conception mechanics
  • Added leisure system — citizens spend free time with partners and friends
  • Modular architecture refactor for CitizenAI system
  • Unified event notification system with deduplication
  • UI refinements: warm earth-tone palette, improved info panels, speed control z-order fix
  • Various bug fixes: building click selection, frozen worker pathing, construction-site pathing, newborn naming

v0.4.0

  • Added milestone system with permanent settlement bonuses and random narrative events
  • Added livestock: chickens and cattle with feeding, breeding, production, and winter exposure
  • Added social buildings: Tavern, Well, and Chapel with happiness auras
  • Added citizen skill progression with XP, 5 levels, and mastery bonuses
  • Added visible crop growth stages replacing flat-rate production
  • Added personality traits (Hardworking, Lazy, Social, etc.) affecting behavior and productivity

v0.3.0

  • Added Bakery building and cooking system with cooked meal buffs (warmth, happiness, energy)
  • Added seasonal festivals at the Town Hall with lingering bonus effects
  • Added Town Hall building

v0.2.0

  • Added save/load system (IndexedDB + localStorage)
  • Added event log panel for settlement notifications
  • Added settings panel
  • UI improvements: HiDPI rendering, camera edge-panning removal

v0.1.0

  • Initial release: ECS architecture, map generation, A* pathfinding
  • Core systems: seasons, citizen AI, construction, production, needs, storage, population, trade, disease, weather
  • 20+ building types across housing, storage, food, resource, services, and infrastructure categories
  • Full day/night cycle with seasonal daylight variation
  • Particle effects: smoke, snow, leaves
